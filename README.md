# consulta-cep-viacep-e-pesquisa-webservice
Site desenvolvido para consultar endereços a partir do CEP utilizando API.

O usuário digita um CEP e o sistema retorna as informações do endereço usando a API ViaCEP.

Funcionalidades
- Entrada de CEP
- Consulta automática via API
- Exibição de endereço

API utilizada
- ViaCEP

Tecnologias
- HTML
- CSS
- JavaScript
